# am2302

Example of usage PIO + DMA in RP2350 (PI PICO2) with humidity and temperature sensor AM2302.

- uC: rp2350
- board: pico2
- sensor: am2302
- communication protocol: one wire 
- used peripherals: PIO + DMA

## How to run this repo

- clone this repo
- open VS Code and install pico addon
- import this project
- compile and run
